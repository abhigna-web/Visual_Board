# **App Name**: Collaborative Canvas

## Core Features:

- User Authentication: Enable user login and registration with secure authentication flows, matching the existing site's user experience.
- Canvas Drawing Tools: Implement all existing drawing tools and canvas functionalities (e.g., pen, eraser, shapes) to allow users to create and edit drawings exactly as they do currently.
- Board Creation & Management: Allow users to create new drawing boards, save their progress, load existing boards, and delete boards, with data persisted online.
- Real-time Collaboration Engine: Enable multiple users to co-edit the same board concurrently, with changes appearing in real-time for all participants, matching existing functionality.
- Shareable Invitation Links: Provide functionality to generate unique, secure sharing links to invite collaborators to specific boards, granting appropriate access as per the existing system.

## Style Guidelines:

- Primary color: A serene and vibrant Blue-Green (#2E95B8) chosen to reflect creativity and clarity for a focal user experience. All color values to precisely match the existing site.
- Background color: A highly desaturated, light Blue-Green (#F0F3F4) providing a clean, spacious canvas that facilitates user focus. All color values to precisely match the existing site.
- Accent color: A lively Spring Green (#63E9C7) to highlight interactive elements and call-to-actions, ensuring visual consistency and functionality. All color values to precisely match the existing site.
- Body and headline font: 'Inter' (sans-serif) chosen for its modern, neutral, and highly readable characteristics, suitable for a functional creative application. Font choices to precisely match the existing site.
- Icons and graphic elements will be preserved exactly as found in the existing website design to ensure full fidelity to the current user interface and experience.
- The page structure, component placement, and overall spatial arrangement will be meticulously replicated to match the existing website's layout precisely.
- All interactive animations, transitions, and user feedback responses will be faithfully recreated to mirror the current website's dynamic behavior.
